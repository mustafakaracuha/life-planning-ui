# Life Planning UI Demo


## Technologies and Libraries

- React Js
- Next Js
- Tailwind CSS
- React Icons

  
## Installation 

```bash 
    git clone https://github.com/mustafakaracuha/life-planning-ui.git
```
```bash 
    cd life-planning-ui
```
```bash 
    npm install
```
```bash 
    npm run dev
```
